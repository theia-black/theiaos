# `theiaos setup`

Initialize `~/.theiaos/theiaos.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
theiaos setup
theiaos setup --workspace ~/.theiaos/workspace
```

To run the wizard via setup:

```bash
theiaos setup --wizard
```
